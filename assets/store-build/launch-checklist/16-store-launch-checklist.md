# BIK Solutions Shop — Complete Store Launch Checklist

*Work through this top to bottom. Tick each item before going live. Items marked [STEVE] require your input or approval.*

---

## PHASE 1: Platform Setup

### Shopify Account
- [ ] Create Shopify account at shopify.com/au — recommend Basic plan ($39 AUD/month) to start
- [ ] Select Australian store currency: AUD
- [ ] Set store timezone: Australia/Brisbane (no daylight saving)
- [ ] Store name: BIK Solutions Shop (or BIK Site Protection)
- [ ] [STEVE] Add business legal name: BIK Solutions Pty Ltd
- [ ] [STEVE] Add ABN in Settings → General

### Domain
- [ ] [STEVE] Decide domain: shop.biksolutions.com.au (subdomain of existing site) OR biksiteprotection.com.au (standalone)
- [ ] Purchase/configure domain (if new domain: ~$20/yr via Shopify or namecheap.com.au)
- [ ] Connect domain to Shopify (Settings → Domains)
- [ ] Set as primary domain
- [ ] Enable SSL (auto via Shopify once domain is connected — verify green padlock)
- [ ] Set up redirect: www → non-www (or vice versa)

## PHASE 2: Email Setup

- [ ] [STEVE] Create professional email: shop@biksolutions.com.au (or hello@biksitations.com.au)
- [ ] Recommended: Google Workspace ($9/month) or Zoho Mail (free tier)
- [ ] Connect email to Shopify (Settings → Notifications → Sender email)
- [ ] Set up email forwarding from Shopify notifications
- [ ] [STEVE] Set up email marketing platform: Klaviyo (free up to 250 contacts) or Shopify Email
- [ ] Create welcome email template
- [ ] Create abandoned cart email sequence (2 emails: 1hr and 24hr)
- [ ] Create post-purchase email (3 days after delivery)
- [ ] Create review request email (7 days after delivery)

---

## PHASE 3: Payment Gateways

- [ ] Enable Shopify Payments (Australian merchant) — includes Visa, Mastercard, AMEX
- [ ] [STEVE] Complete identity verification for Shopify Payments (requires ABN, bank account details)
- [ ] Connect Australian bank account for payouts
- [ ] Enable PayPal (Settings → Payments → Alternative payment methods)
- [ ] Enable Afterpay (via Shopify Payments or Afterpay direct — requires application)
  - [ ] Apply at afterpay.com/en-AU/business
  - [ ] Once approved, connect via Settings → Payments
- [ ] Enable Zip Pay (similar application process via zip.co/au/business)
- [ ] Enable Apple Pay / Google Pay (auto via Shopify Payments)
- [ ] Test a purchase with a test payment before launch

---

## PHASE 4: Shipping Settings

- [ ] Settings → Shipping and delivery
- [ ] Set shipping from address: [STEVE — add warehouse/office address]
- [ ] Configure shipping zones:
  - Australia — Flat rate $15 (under $150)
  - Australia — Free shipping on orders $150+ (weight/price-based rule)
  - Oversized / freight — Configure at checkout or manual quote for board packs
- [ ] Set up weight-based rules if applicable (boards vs. rolls vs. tape)
- [ ] [STEVE] Confirm courier/freight partner:
  - Option A: Australia Post (easy setup via Shopify integration)
  - Option B: Sendle (competitive rates, integrates with Shopify)
  - Option C: Couriers Please (good for QLD)
  - Option D: Supplier ships direct (confirm dropship terms with Protecta/ProTrade)
- [ ] Install relevant courier app (Shopify App Store: Sendle, Australia Post, etc.)
- [ ] Test shipping rate calculation on a sample order
- [ ] Enable "Show estimated delivery dates" at checkout

---

## PHASE 5: Tax Settings

- [ ] Settings → Taxes and duties
- [ ] Set up Australia as tax region
- [ ] Set Australian GST at 10%
- [ ] Enable "Include tax in prices" OR "Show prices excluding tax with tax shown at checkout" — **recommend the latter for B2B/trade buyers**
- [ ] [STEVE] Confirm with accountant: Are all products standard-rated for GST? (Yes — physical goods in Australia are GST-applicable)
- [ ] Set up Business Activity Statement (BAS) reporting — recommend Xero or MYOB integration with Shopify

## PHASE 9: Analytics & Tracking

### Google Analytics 4
- [ ] Create GA4 property at analytics.google.com
- [ ] [STEVE] Add GA4 measurement ID to Shopify (Online Store → Preferences → Google Analytics)
- [ ] Or install via Google & YouTube Shopify app (easier)
- [ ] Enable ecommerce tracking
- [ ] Verify GA4 is receiving data (Realtime report)

### Google Search Console
- [ ] Go to search.google.com/search-console
- [ ] Add property for your domain
- [ ] Verify via domain or HTML tag (Shopify: paste verification tag in theme code)
- [ ] Submit sitemap: yourdomain.com/sitemap.xml
- [ ] Verify sitemap is processing

### Google Merchant Centre
- [ ] Create account at merchants.google.com
- [ ] [STEVE] Verify business: ABN, business address, phone
- [ ] Add website and verify/claim
- [ ] Connect Shopify product feed (Google & YouTube app on Shopify — free, syncs product catalogue)
- [ ] Set up free product listings (Google Shopping — free traffic)
- [ ] Fix any feed errors flagged in Merchant Centre

### Meta Pixel (Facebook/Instagram)
- [ ] Create a Meta Business Manager account (business.facebook.com) if not already done
- [ ] Create a Meta Pixel in Events Manager
- [ ] Install pixel on Shopify via Facebook & Instagram app (Shopify App Store — free)
- [ ] Verify pixel is firing on product pages, add-to-cart, checkout and purchase events
- [ ] Set up Conversions API (done via the same Meta app — improves data quality)
- [ ] Create a custom audience: "Visited site in last 30 days" for retargeting later

### Google Ads Conversion Tracking
- [ ] Create Google Ads account (if not already done) — link to GA4 is preferred method
- [ ] In GA4: Mark "Purchase" event as a conversion
- [ ] Import GA4 purchase conversion into Google Ads
- [ ] Or set up Google Ads conversion tag directly (Shopify → Additional scripts section in checkout settings)
- [ ] Verify conversion is firing on a test purchase

## PHASE 12: Launch & Post-Launch

### Launch day
- [ ] Remove password from store (Online Store → Preferences → uncheck "Enable password")
- [ ] Post on LinkedIn/Facebook announcing the store
- [ ] [STEVE] Text/email existing builder contacts with the launch announcement and a discount code
- [ ] Set up a launch offer (e.g., 10% off first order with code LAUNCH10 — 7-day expiry)
- [ ] Verify all tracking is live (GA4 Realtime, Meta Pixel Helper extension)

### First week
- [ ] Monitor Google Search Console for any crawl errors
- [ ] Monitor Shopify analytics for traffic and conversion
- [ ] Set up weekly Google Ads budget: recommend starting with $20–$30/day on Shopping ads
- [ ] Submit Google Merchant Centre product feed (if not auto-syncing)
- [ ] [STEVE] Send launch email to newsletter list

### Backup
- [ ] Export product CSV from Shopify (Products → Export) — store copy offline
- [ ] Document all API keys, app settings and integrations in a secure password manager
- [ ] [STEVE] Add Steve's personal email as a staff account in Shopify
