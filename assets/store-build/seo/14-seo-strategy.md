## Technical SEO Checklist

- [ ] Custom domain connected (biksolutions.com.au or shop.biksolutions.com.au)
- [ ] SSL certificate active (auto via Shopify)
- [ ] Sitemap submitted to Google Search Console
- [ ] Robots.txt configured (Shopify default is fine)
- [ ] All product images have descriptive alt text
- [ ] All pages have unique SEO titles and meta descriptions
- [ ] Canonical URLs set on variant pages
- [ ] Page speed optimised (compress images, use WebP format)
- [ ] Schema markup: Product schema on all product pages (Shopify handles basic; add Review schema when reviews collected)
- [ ] Google Analytics 4 connected
- [ ] Google Search Console verified
- [ ] Google Merchant Centre feed set up (for Shopping ads)
