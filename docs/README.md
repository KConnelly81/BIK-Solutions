## What This Repository Is

This repository contains the complete operating manual for BIK Solutions — a trades services business evolving into an AI-powered Business Operating System for Australian construction companies.

It holds both the public-facing website and all internal product, business, and technical documentation.

## How to Use This Documentation

- **New team member?** Start with `product-vision.md`, then `technical-architecture.md`.
- **Working on a feature?** Check `feature-backlog.md` and the relevant spec in `specifications/`.
- **Making a significant decision?** Record it in `decisions/` using the ADR format.
- **Updating the roadmap?** Edit `product-roadmap.md` and `feature-backlog.md` together.

## Documentation Standards

Every document must include: Purpose, Last Updated, Status (Draft / Active / Archived), Owner.

Keep documentation concise and cross-referenced. No duplicate documents — one source of truth per topic.
